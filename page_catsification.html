Catsification

Project Status Completed
Project Type Gamejam
Primary Role(s) Programmer
Project Duration 3 days
Engine and Language Unity, C#

About the project

Casual game for WebGL where the player memorizes the fur pattern of several cats and places them on their respective baskets.

This project was developed and submited as an entry for the Mermelada Jam on Itch.io, ranked 15th from the 63 entries.


Gameplay

Take notice of each cats fur pattern for a few seconds, once the lights are out drag and drop each cat to a basket.

Once all cats are placed, you will advance to the next level, given aditional 20 seconds to the timer.

For each wrong placed cat, 2 seconds will be distounted from the timer, once the time reach 0, the game will end.
