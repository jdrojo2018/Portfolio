Spacecraft Protector

Project Status Completed
Project Type University
Project Duration ~1 week
Engine and Language Java

About the project

Simple 2D space shooter for Android maded with a custom engine.

Most assets come from Turret Protector , my first major personal project.


Gameplay and Features

  • There are 2 playable ships, each one with different main weapons.
  • Each weapon has 4 levels of upgrade.
  • There are 3 types of enemies with different behaviors.
  • There are 3 difficulty modes.

In order to win, the player has to survive 15 waves of enemies.
